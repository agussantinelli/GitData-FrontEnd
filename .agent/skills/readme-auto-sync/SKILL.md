# 🔄 GitData README Auto-Sync

Guidelines for keeping the README synchronized with the project evolution.

## 📏 Rules
1. **Estructura del Proyecto**:
   - **CUÁNDO**: Cada vez que crees, muevas o elimines un archivo (`.ts`, `.vue`, `.scss`) o un directorio.
   - **QUÉ HACER**: Actualizar la sección "Project Structure" en el `README.md` (formato HTML/pre para GitData).
2. **Dashboard Status**:
   - **CUÁNDO**: Al completar nuevas funcionalidades de la "Propuesta de Valor".
   - **QUÉ HACER**: Actualizar el estado en el README.
